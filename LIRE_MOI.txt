 POUR CRÉER L'EXÉCUTABLE WINDOWS (.exe)
═══════════════════════════════════════════════════════════════

ÉTAPE 1 - Installer Node.js (si pas déjà fait)
  → Aller sur : https://nodejs.org
  → Télécharger la version "LTS" (ex: 20.x LTS)
  → Installer en laissant toutes les options par défaut

ÉTAPE 2 - Compiler l'application
  → Double-cliquer sur : COMPILER_EXE.bat
  → Attendre 3-5 minutes (téléchargement d'Electron la 1ère fois)
  → L'EXE sera créé dans le dossier "dist\"

ÉTAPE 3 - Utiliser
  → dist\MRDPSTOCK Setup X.X.X.exe  → Installeur (recommandé)
  → dist\MRDPSTOCK X.X.X.exe        → Portable (sans installation)

═══════════════════════════════════════════════════════════════
 TESTER SANS COMPILER (mode développement)
═══════════════════════════════════════════════════════════════

  1. Ouvrir un terminal dans ce dossier
  2. Exécuter : npm install
  3. Exécuter : npm start
